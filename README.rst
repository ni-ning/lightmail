安装
********

目前 lightmail 支持的 Python 环境有 2.7, 3.4, 3.5, 3.6, 3.7, 3.8。


为了简化安装过程，推荐使用 pip 进行安装

.. code-block:: bash

    pip install lightmail

升级 exmail-sdk 到新版本::

    pip install -U lightmail

如果需要安装 GitHub 上最新代码::

    pip install https://github.com/ni-ning/lightmail/archive/master.zip
